# 🌐 Network Basics - ქსელის საფუძვლები

ეს არის საგანმანათლებლო ვებ-პროექტი, რომელიც შექმნილია კომპიუტერული ქსელების ფუნდამენტური პრინციპების ასახსნელად. საიტი მოიცავს ინფორმაციას ძირითადი პროტოკოლებისა და მისამართების სისტემების შესახებ.

🌐 **Live Demo:** [dapper-yeot-062887.netlify.app](https://dapper-yeot-062887.netlify.app/)

## 📚 რას ისწავლით ამ საიტზე?

პროექტი დაყოფილია ლოგიკურ სექციებად:
- **HTTP & HTTPS:** ინფორმაცია მონაცემთა გადაცემის პროტოკოლებსა და უსაფრთხოებაზე.
- **DNS (Domain Name System):** როგორ გარდაიქმნება დომენის სახელები IP მისამართებად.
- **IP Basics:** ინტერნეტ პროტოკოლის მისამართების სტრუქტურა და დანიშნულება.

## ✨ ფუნქციები

- **Structured Content:** ინფორმაცია წარმოდგენილია გასაგები და ორგანიზებული სახით.
- **Navigation Menu:** მარტივი გადასვლა სხვადასხვა სასწავლო თემებს შორის.
- **Educational UI:** სუფთა და აკადემიური დიზაინი, რომელიც ხელს უწყობს მასალის ათვისებას.

## 🛠️ ტექნოლოგიური მხარე

- **HTML5:** სემანტიკური სტრუქტურისთვის.
- **CSS3:** ლეაუთის და ვიზუალური სტილისთვის.
