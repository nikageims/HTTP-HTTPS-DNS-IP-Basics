რა არის DNS?

DNS (Domain Name System) გარდაქმნის დომენის სახელს IP მისამართად.

  • მაგ: google.com
  • ადამიანისთვის მარტივი
  • კომპიუტერისთვის საჭირო
🌐 DNS

რა არის IP?

IP მისამართი არის უნიკალური ნომერი, რომელიც თითოეულ მოწყობილობას აქვს ინტერნეტში.

  • მაგ: 192.168.1.1
  • მაგ: 8.8.8.8
  • უნიკალური იდენტიფიკატორი
📡 IP

3️⃣ სერვერთან კავშირი
4️⃣ საიტი იტვირთება
