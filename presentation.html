HTTP & HTTPS

რა არის და რა განსხვავებაა მათ შორის

რა არის HTTP?

HTTP (HyperText Transfer Protocol) არის პროტოკოლი, რომელიც ბრაუზერსა და სერვერს შორის მონაცემების გადაცემას უზრუნველყოფს.

  • არ არის დაცული
  • მონაცემები იგზავნება ღიად
  • გამოიყენებოდა ძველ საიტებზე

რა არის HTTPS?

HTTPS (HyperText Transfer Protocol Secure) არის HTTP-ის უსაფრთხო ვერსია, რომელიც იყენებს შიფრაციას.

  • დაცულია SSL/TLS-ით
  • მონაცემები დაშიფრულია
  • გამოიყენება თანამედროვე საიტებზე

HTTP და HTTPS განსხვავება

HTTP HTTPS
არ არის უსაფრთხო დაცულია
მონაცემები ღიაა მონაცემები დაშიფრულია
არ აქვს სერტიფიკატი აქვს SSL სერტიფიკატი

რა არის SSL / TLS?

SSL (Secure Sockets Layer) და TLS (Transport Layer Security) არის უსაფრთხოების ტექნოლოგიები, რომლებიც მონაცემებს შიფრავს.

  • იცავს პაროლებს
  • იცავს საბანკო მონაცემებს
  • არ აძლევს ჰაკერს ინფორმაციის წაკითხვის საშუალებას

როგორ მუშაობს Request / Response?

როცა მომხმარებელი შედის ვებსაიტზე, ბრაუზერი აგზავნის მოთხოვნას (Request) სერვერზე. სერვერი პასუხობს (Response) და აბრუნებს საიტის კონტენტს.

1️⃣ ბრაუზერი აგზავნის მოთხოვნას
2️⃣ სერვერი ამუშავებს მონაცემებს
3️⃣ სერვერი აბრუნებს პასუხს

HTTPS და Google (SEO)

Google უპირატესობას ანიჭებს HTTPS საიტებს. ასეთი საიტები უკეთ ჩანს ძიების შედეგებში.

სად გამოიყენება HTTPS?

  • ონლაინ მაღაზიები
  • ბანკის ვებსაიტები
  • სოციალური ქსელები
  • ელფოსტა
  • სახელმწიფო პორტალები

რა საფრთხეა HTTP-ის გამოყენებისას?

  • პაროლების მოპარვა
  • პირადი ინფორმაციის გაჟონვა
  • ჰაკერული შეტევები
  • ფიშინგი
